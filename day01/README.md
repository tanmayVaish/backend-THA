# Day 1

## Redis

### Installation

on Ubuntu:

```bash
sudo apt update
sudo apt install redis
sudo service redis start
```

On Arch:
```bash
yay -S redis
sudo systemctl start redis
```

On Windows:

[Download Latest MSI Here](https://github.com/redis/redis/releases)

```
open -> next -> next -> next -> install -> done
```

### Usage

```bash
redis-cli
```
